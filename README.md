# pt1-e04-la-mere

> A Vue.js project

## Requirements
Node js > 5.4.4

## Build Setup


``` bash
# clone repository
git clone https://github.com/nothaldir/PT1-E04-LA-MERE
cd PT1-E04-LA-MERE

# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build
```
